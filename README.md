# 🎓 Student Management System

A command-line Student Management System built with Python to practice **Object-Oriented Programming (OOP)**, **Classes**, and **CRUD (Create, Read, Update, Delete)** operations.

---

## 🚀 Features

- Add Student
- View All Students
- Update Student Information
- Delete Student Records
- Search Student by ID
- Menu-Driven Interface
- Object-Oriented Design using Classes

---

## 📚 Skills Practiced

- Object-Oriented Programming (OOP)
- Classes and Objects
- Constructors (`__init__`)
- Object Attributes
- Lists of Objects
- Functions
- Loops
- Conditional Statements
- CRUD Operations
- User Input Handling

## ▶️ How to Run

1. Clone the repository.

2. Open the project folder.

3. Run the program:

```bash
python student_management.py
```
